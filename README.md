# yourpal
A web application to manage a Discord bot.

## Installation
Download with https
```bash
git clone https://github.com/coolfriends/yourpal.git
```

or SSH
```bash
git clone git@github.com:coolfriends/yourpal.git
```

And install packages
```
npm install
```

## Start
Run the development webserver
```bash
npm start
```


## Tooling
This project was created with create-react-app.
